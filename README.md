# Employee Management SQL

## Overview
This project focuses on employee data analysis using SQL.

It includes SQL queries to analyze:
- Employee records
- Department information
- Salary trends
- Department-wise summaries

## Files Included

- `employees.csv` – Employee details
- `departments.csv` – Department information
- `employee_analysis.sql` – Employee-based SQL analysis
- `salary_analysis.sql` – Salary-related SQL queries
- `department_summary.sql` – Department summary analysis

## Skills Used
- SQL
- Joins
- Aggregate Functions
- GROUP BY
- ORDER BY
- Business Data Analysis

## Purpose
To practice SQL queries using employee and department datasets in a real-world business scenario.
